# desafio-tetris-stack
Desafio Tetris Stack: Estruturas de Dados em C  Este repositório contém a implementação do Desafio Tetris Stack, uma atividade aplicada na aplicação de estruturas de dados fundamentais na linguagem C, utilizando fila, pilha e a integração entre elas para simular o gerenciamento de peças do jogo Tetris Stack.
